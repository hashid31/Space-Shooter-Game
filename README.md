## Setup Instructions

### 1. Clone the Repository

```
git clone https://github.com/hashid31/space-shooter-game.git
cd space-shooter-game
```

### 2. Install Dependencies

Make sure Python and pip are installed, then:

```
pip install pygame
```

### 3. Run the Game

```
python spaceshooter.py
```
